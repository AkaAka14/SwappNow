# SwappNow

A peer-to-peer skill exchange platform (MERN) with AI-driven matchmaking (DeepSeek), real-time collaboration (Socket.io), and vector search powered recommendations.



## About

SwappNow is a full-stack peer-to-peer skill exchange platform that helps users teach and learn skills through AI-powered matchmaking and live collaboration tools. Matches are generated using DeepSeek and MongoDB vector search to provide personalized recommendations. Real-time messaging, session scheduling, and live collaboration are implemented with Socket.io.

## Key Features
- AI Skill Matching Agent (DeepSeek + MongoDB vector search)
- Real-time 1:1 messaging and collaboration (Socket.io)
- Session scheduling and notifications
- User profiles, skill listings, ratings & reviews
- RESTful APIs for client-server communication
- Scalable architecture design with Redis adapter support for Socket.io
