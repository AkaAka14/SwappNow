# SwappNow

A peer-to-peer skill exchange platfor with AI-driven matchmaking and vector search powered recommendations.



## About

SwappNow is a full-stack peer-to-peer skill exchange platform that helps users teach and learn skills through AI-powered matchmaking and live collaboration tools. Matches are generated and MongoDB is used for vector search to provide personalized recommendations. Real-time messaging, session scheduling..

## Key Features
- AI Skill Matching Agent (LLM + MongoDB vector search)
- Session scheduling and notifications
- User profiles, skill listings, ratings & reviews
